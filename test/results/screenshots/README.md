# Screenshots Directory

Place screenshots of test reports here for documentation and submission.

## Required Screenshots

1. **Code Coverage Report** (`coverage-report.png`)
   - Run: `npm run test:coverage`
   - Open: `coverage/index.html`
   - Screenshot: Overall coverage summary

2. **Mutation Testing Report** (`mutation-report.png`)
   - Run: `npm run mutation`
   - Open: `reports/mutation/index.html`
   - Screenshot: Mutation score dashboard

3. **Test Execution** (`test-execution.png`)
   - Run: `npm test`
   - Screenshot: Console output showing all tests passing

4. **Integration Tests** (`integration-tests.png`)
   - Run: `npm run test:integration`
   - Screenshot: Integration test results

## File Naming Convention

- `coverage-report.png` - Overall coverage
- `coverage-validators.png` - Validators module coverage
- `coverage-pricing.png` - Pricing module coverage
- `mutation-report.png` - Mutation testing dashboard
- `mutation-validators.png` - Validators mutations
- `mutation-pricing.png` - Pricing mutations
- `test-execution.png` - All tests passing
- `integration-tests.png` - Integration tests

## For Submission

Include these screenshots in your project report to demonstrate:
1. Comprehensive test coverage
